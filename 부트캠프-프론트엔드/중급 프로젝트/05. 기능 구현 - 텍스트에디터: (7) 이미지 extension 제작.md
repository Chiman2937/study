# 📝 이미지 Extension 제작하기

## ❓ Tiptap 기본 이미지 Extension의 문제점
Tiptap이 제공하는 이미지 Extension은 다음과 같은 문제점이 있었다.

1. Editor에서 이미지를 선택해도 별도의 focus style이 적용되지 않는다.
2. 이미지 Resize를 지원하지 않는다.
3. 이미지 좌/우 정렬을 지원하지 않는다.

## 🔍 구조 설계
