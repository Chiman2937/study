# 📝 Formatting Context

<br></br>
## ✅ Formatting Context란?
- 요소들이 어떻게 배치될 지 결정하는 레이아웃 규칙

<br></br>

## ✅ Formatting Context의 종류

|종류|설명|
|---|---|
|BFC(Block Formatting Context)|요소들을 배치하는 방식이자 독립된 공간|
|IFC(Inline Formatting Context)|인라인요소들을 배치하는 방식|

<br></br>
### ※ 비공식 용어(참고용)

- 아래의 4가지는 모두 BFC에 포함되어있는 개념이다.
